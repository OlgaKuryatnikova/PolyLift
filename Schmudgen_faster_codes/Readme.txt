These are the versions of the Schmudgen non-SOS certificates with faster construction times (solution times are approximately the same as in Yalmip). 
Here we code the polynomials using some functions from SOSTOOLS version 1.00 (first reference below) and DIGS (second reference below) instead of sdpvar.
